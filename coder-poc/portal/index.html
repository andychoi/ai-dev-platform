Quick Start

  1. Open Coder WebIDE and accept the self-signed certificate warning
  2. Click Login with OIDC and sign in with a test user (e.g. contractor1 / password123)
  3. Create a workspace from the contractor-workspace template
  4. Click code-server to open VS Code in the browser
  5. Open Roo Code in the sidebar — AI chat is pre-configured via LiteLLM
